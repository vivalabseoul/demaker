# 문서 (Documentation)

이 폴더에는 프로젝트 관련 문서들이 정리되어 있습니다.

## 📚 문서 목록

- **[PERFORMANCE_ANALYSIS.md](./PERFORMANCE_ANALYSIS.md)** - 성능 분석 및 개선 방안
- **[Guidelines.md](./Guidelines.md)** - 디자인 및 개발 가이드라인
- **[Attributions.md](./Attributions.md)** - 라이선스 및 출처 정보

## 📝 문서 작성 규칙

앞으로 생성되는 모든 마크다운(.md) 파일은 이 `docs/` 폴더에 저장됩니다.

단, 프로젝트 루트의 `README.md`는 예외로 프로젝트 루트에 유지됩니다.
